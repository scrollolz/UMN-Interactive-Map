# UMN Interactive Map

The UMN-TC Interactive Map intends to consolidate existing maps (gopherway, microwaves, etc.) into one interactive map with togglable filters for the different locations.

## Tech Stack
Leaflet, HTML, CSS, JS, JQuery, Firebase
